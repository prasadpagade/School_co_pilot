# Screenshots Directory

Place app screenshots here.

## How to Add Screenshots

1. **Take a screenshot** of the app:
   - Desktop: Cmd+Shift+4 (Mac) or Snipping Tool (Windows)
   - iPhone: Screenshot button
   - Use the app at: `http://localhost:8000` or your deployed URL

2. **Save the screenshot** as `app-screenshot.png` in this directory

3. **Recommended sizes**:
   - Desktop: 1920x1080 or 1280x720
   - Mobile: 750x1334 (iPhone) or 1080x1920 (Android)

4. **Screenshot should show**:
   - Chat interface
   - Example question/answer
   - Mobile-responsive design (optional)

## Alternative: Add Screenshot from GitHub

You can also:
1. Upload the screenshot directly to GitHub
2. Get the raw image URL
3. Update README.md with the URL
