## Welcome!

This app helps you stay organized with school information. Ask questions, get answers, and manage your calendar - all in one place!

## Getting Started

1. **Open the app** in your web browser (works on phone, tablet, or computer)
2. **No login required** - just start asking questions!

### üí¨ Chat Tab

**Example questions:**
- "What events are happening this week?"
- "When is the next field trip?"
- "What is the dress code policy?"

**Tips:**
- Use the microphone button to ask questions by voice

### üìÖ Calendar Tab
Upload screenshots of conversations to automatically add events to your calendar.

**How to use:**
1. Take a screenshot of a text conversation (SMS, WhatsApp, etc.)
2. Click the image icon or go to Calendar tab
3. Upload the image
4. Review extracted events
5. Click "Add to Calendar" for each event

- See when emails were last checked

### üìä Metrics Tab

## Voice Commands

You can use voice to:
- Ask questions: Click mic, speak your question
- Add calendar events: Say "Add Music class on December 15th at 3pm"

## Calendar Integration

**First-time setup:**
1. The app will prompt you to authorize Google Calendar access
2. Click "Allow" when prompted
3. You only need to do this once

**Adding events:**
- Upload conversation images (Calendar tab)
- Use voice commands: "Add [event] on [date] at [time]"
- Click "Add to Calendar" buttons in chat responses

## Tips & Tricks

1. **Bookmark the app** - Add to home screen for easy access
2. **Check regularly** - New emails are automatically processed daily
3. **Use voice** - Faster than typing on mobile
4. **Upload images** - Quick way to add multiple events at once

## Troubleshooting

**App not loading?**
- Check your internet connection
- Try refreshing the page
- Clear browser cache if needed

**Can't add calendar events?**
- Make sure you've authorized Google Calendar access
- Check the Schedule tab for calendar status

**Questions not working?**
- Make sure emails have been ingested
- Try rephrasing your question
- Check the Schedule tab to see last email check time

## Need Help?

If you encounter issues:
1. Check the Schedule tab for system status
2. Try refreshing the page
3. Contact the app administrator
